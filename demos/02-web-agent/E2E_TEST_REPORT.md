npx playwright test e2e/auth.spec.ts

npm run test:e2e:debug

## 已知问题和修复建议

### 问题 1: 注册后跳转超时
**原因**: 可能是路由跳转延迟或后端响应慢
**修复**: 增加超时时间，或使用 `waitForNavigation`

### 问题 2: 退出登录按钮定位
**原因**: 圆形按钮选择器不够精确
**修复**: 使用更具体的选择器，如 `[aria-label="退出登录"]`

### 问题 3: 聊天功能测试超时
**原因**: SSE 流式响应可能需要更长时间
**修复**: 增加超时时间，或优化等待逻辑
